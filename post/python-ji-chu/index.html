python基础

变量和简单数据类型

变量

变量的命名和使用

  1. 变量名只能包含字母、数字和下划线。
  2. 变量名不能包含空格，但可使用下划线来分隔其中单词
  3. 关键字和函数名不可以用作变量名
  4. 变量名尽量简短又有描述性
  5. 慎用小写‘l’，和大写‘O’，容易被看错

字符串

就是一系列字符。用引号括起的都是字符串。
（可以是单引号也可以是双引号）

使用方法修改字符串的大小写

对于字符串，可执行的最简单的操作之一是修改其中单词的大小写。

  1. .title（）以首字母大写的方式显示每个单词，即是将每个单词的首字母都改为大写。
  2. .lower()全小写显示每个单词
  3. .upper全大写显示每个单词

合并（拼接字符串）

python使用“+”来合并字符串
first_name = "Ada"
last_name = "lovelace"
full_name = first_name + " " + last_name

拼接创建信息后，可以把整条消息存储在一个变量中：
message = "Hello, " + full_name.title() + "!"
print(message)

使用制表符或换行符来添加空白

在编程中，空白泛指任何非打印字符，如空格、制表符、换行符。你可使用空白来组织输出，使其易读。

  • 要在字符串中添加制表符，可使用字符组合\t
  • 要在字符串中添加制表符，可使用字符组合\n

删除空白

有空白的字符串和无空白的字符串是两个不同的字符串。

Python能找出字符串开头和末尾多余的空白。要确保字符串末尾没有空白，可使用方法rstrip（）

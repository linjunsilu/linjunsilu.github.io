Java的基本数据类型

标识符

用来标识类名、变量名、方法名、类型名、数组名、文件名的有效字符序列称为标识符。
简而言之，标识符就是一个名字。

标识符的语法规则

  1. 标识符由字母、下划线、美元符号、数字组成，长度不受限制
  2. 标识符的第一个字符不能是数字字符
  3. 标识符不能为关键字
  4. 标识符不能是true、false和null

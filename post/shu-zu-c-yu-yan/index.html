数组（C语言）

一维数组

一维数组的定义

类型说明符 数组标识符[常量表达式]；
eg.
int iArray[5];

一维数组的引用

数组标识[下标]
eg.
iArray[2];

一维数组初始化

①直接对数组元素赋初值
eg1.
int iArray[6]={1,2,3,4,5,6};
②只给一部分元素赋值，其他元素默认为0
eg2.
int iArray[6]={0,1,2};
③不指定数组长度，直接赋值，赋值个数即是数组元素个数即是数组长度
eg3.
int iArray[]={1,2,3,4};

二维数组

二维数组的定义

数据类型 数组名[常量表达式 1][常量表达式 2];
eg.
int array[3][4];

二维数组的引用

数组名[下标][下标];
eg.
array[2][3];

二维数组的初始化

①可以将数据写在大括号里，按照数组元素排列顺序对元素赋值。
eg.
int array[2][2]={1,2,3,4};
②在为所有的元素赋值时，可以省略行下标，但是不能省略列下标。
eg.
int array[][3]={1,2,3,4,5,6};
③也可以分行给数组元素赋值。
eg.
int a[2][3]={{1,2,3},{4,5,6}};
在分行赋值的时，可以只对部分元素赋值。
eg.
int a[2][3]={{1,2},{4,5}};
④二维数组也可以直接对数组元素赋值。
eg.
int a[2][3];
a[0][0]=1;
a[0][1]=2;

二维数组的应用

eg.求对角线之和

字符数组

字符数组的定义

char 数组标识符[常量表达式]
eg.
char cArray[5];

字符数组的引用

eg.
cArray[0]='H';
cArray[0]='e';
cArray[0]='l';
cArray[0]='l';
cArray[0]='o';
